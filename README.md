# Cadastro de Usuário

![Captura de tela do Cadastro de Usuário](https://github.com/jessica-sobreira/cadastrando_usuario_recado/assets/117686537/c6c58f98-0422-4451-a451-521c1e39e80b)

Esta é uma prática de cadastro de usuários e recados realizada durante a aula sobre formulários da Growdev. O objetivo dessa atividade é aplicar e consolidar conhecimentos relacionados à criação de formulários em HTML.

## Detalhes da Prática

- **Cadastro de Usuários e Recados:** A prática envolve a criação de um formulário para cadastrar usuários e recados.
- **Experiência em Formulários:** A aula proporcionou uma experiência prática para lidar com formulários, entradas de usuário e manipulação de dados.
